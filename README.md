# 🧟 Zombie Survival 2D

**Zombie Survival** — це динамічний 2D top-down шутер, розроблений на рушії Unity. Гравець має виживати проти хвиль зомбі, кажанів та босів, заробляти монети і купувати нову потужну зброю.

## ✨ Основні особливості
- **Twin-Stick керування:** Гра автоматично розпізнає платформу. На ПК використовується класичне керування (W/A/S/D + Мишка), а на Android вмикаються віртуальні джойстики (від Fenerax Studios).
- **Система хвиль та босів:** Кожен рівень має кілька хвиль ворогів. На останніх хвилях гравця чекають епічні битви з босами (наприклад, Boss Golem), які мають власну велику смужку здоров'я.
- **Внутрішньоігрова крамниця:** Вороги залишають після себе монети. За ці монети у Головному Меню можна розблоковувати нову зброю (пістолет, дробовик, автомат).
- **Система прогресу:** Після проходження рівня гравець отримує від 1 до 3 зірок залежно від результату. Прогрес, зірки та зароблені монети назавжди зберігаються між сесіями.
- **Аудіо менеджмент:** Налаштування музики та звукових ефектів зберігаються і автоматично застосовуються на всіх сценах.

## 📱 Як пограти на телефоні (Android)
Готовий для встановлення файл гри (`.apk`) знаходиться у папці **`Android APK`** в корені проєкту. 
Просто перенесіть цей файл на свій Android-пристрій і натисніть на нього, щоб встановити. 

## 🛠 Технічні деталі (Для розробників)
- **Рушій:** Unity
- **Архітектура:** Використовується патерн Singleton для основних менеджерів (`WaveManager`, `UIManager`, `SaveManager`, `CoinManager` тощо).
- **Оптимізація (Object Pooling):** Для ворогів (`EnemyPool`) та куль (`BulletPool`) використовується пулінг об'єктів замість постійного створення та знищення (`Instantiate/Destroy`), що гарантує високу продуктивність навіть на слабких мобільних пристроях.
- **Збереження:** Використовується `PlayerPrefs` для збереження прогресу гравця (відкриті рівні, придбана зброя, кількість монет).

## 🎮 Керування
### Windows / Mac / Linux (ПК)
- **Рух:** `W`, `A`, `S`, `D`
- **Прицілювання:** Курсор мишки
- **Стрільба:** Ліва кнопка мишки (ЛКМ)

### Android / iOS (Мобільні)
- **Рух:** Лівий віртуальний джойстик на екрані
- **Прицілювання та стрільба:** Правий віртуальний джойстик (стрільба починається автоматично при відхиленні джойстика від центру)
